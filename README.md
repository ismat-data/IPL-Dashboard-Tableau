# 🏏 IPL 2008–2020 Tableau Dashboard

## 📊 Project Overview

This project presents an **interactive Tableau dashboard analyzing the Indian Premier League (IPL) from 2008 to 2020**.
The dashboard provides insights into tournament performance, player achievements and match trends using visually engaging charts and cricket-themed visuals.

The objective of this project is to **transform raw IPL match data into meaningful visual insights** that help understand team performance, batting achievements and the impact of toss decisions on match outcomes.

This project demonstrates skills in **data visualization, storytelling with data, dashboard design and analytical thinking using Tableau.**

---

# 🎯 Key Objectives

* Analyze IPL match trends from **2008–2020**
* Identify **top performing batsmen and bowlers**
* Examine **toss decision impact on match results**
* Track **tournament statistics such as total fours and sixes**
* Create an **interactive season selector for dynamic analysis**

---

# 📈 Dashboard Features

## 1️⃣ Orange Cap Analysis

Displays the **highest run scorer of each season** along with visual representation.

Insights include:

* Season-wise top run scorers
* Batting dominance trends
* Visual highlight of Orange Cap winners

---

## 2️⃣ Purple Cap Analysis

Shows the **highest wicket taker of each IPL season**.

Insights include:

* Top bowlers across seasons
* Bowling performance comparison
* Purple Cap winners visualization

---

## 3️⃣ Toss Decision vs Match Result

Analyzes how toss decisions affect match outcomes.

Key metrics:

* Winning percentage after choosing **bat or field**
* Strategic impact of toss decisions
* Comparison of match results based on toss choice

---

## 4️⃣ Matches Won Based on Toss Decision

Shows the number of matches won after choosing:

* Batting first
* Bowling first

This visualization highlights **team strategies and success rates**.

---

## 5️⃣ Tournament Boundaries Analysis

### 🔹 Total Fours

Displays the **total number of fours hit in the tournament**.

### 🔹 Total Sixes

Shows the **total number of sixes hit across the seasons**.

These metrics help analyze **batting aggression and scoring patterns**.

---

## 6️⃣ IPL Winner Highlight

Displays the **season winner** with team logos and trophy visuals.

This section provides a quick summary of **championship teams across IPL seasons**.

---

## 7️⃣ Interactive Season Filter

The dashboard includes a **Season Selector**, allowing users to:

* Filter statistics by season
* Compare performance across years
* Dynamically update all visualizations

---

# 🚀 How to Use the Dashboard

1. Open the `.twbx` file using **Tableau Desktop**.
2. Navigate to **Dashboard 1**.
3. Use the **Season filter** to explore different years.
4. Interact with visualizations to gain insights into IPL performance.
